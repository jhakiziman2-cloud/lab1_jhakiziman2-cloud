HOW TO NAVIGATE THE PROGRAM
1. There are 2 files here as you can see and they are grade-evaluator.py and organizer.sh

To navigate the program a file with the csv extension must be createde forexample grades.csv
After its creation make sure that it is in the same directory as the other two files mentioned above if the program is being ran in a linux terminal and if not, maybe in vscode, make sure that the files are in the same folder

Now if those conditions are met in your linux terminal run the grade-evaluator.py(make sure you have python3 in your device) by typing ./grade-evaluator.py. You will be prompted to write the name of the csv file and get the grades

After that run the organizer.sh file by typing bash organizer.sh to archive the grades and create a time stamp of when the grades were archived
